# LlanquihueTourApp

## Descripción
Sistema de gestión de tours turísticos para la agencia **Llanquihue Tour**,
ubicada en la comuna de Llanquihue, Región de Los Lagos. Lee datos desde
un archivo `.txt` y los almacena en un `ArrayList` de objetos `Tour`,
aplicando recorrido, filtrado y búsqueda sobre la colección.

## Estructura de carpetas
src/main/java/

├── model/    → Tour.java n\
├── data/     → GestorDatos.java n\
└── ui/       → Main.java n\
src/main/resources/ n\
└── tours.txt

## Instrucciones para ejecutar
1. Clona el repositorio: `git clone https://github.com/Mrtucco/llanquihue-tour.git`
2. Abre el proyecto en IntelliJ IDEA Community.
3. Marca la carpeta `resources/` como **Resources Root**.
4. Ejecuta la clase `ui/Main.java`.
